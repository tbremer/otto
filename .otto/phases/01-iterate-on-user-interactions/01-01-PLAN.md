Create a new template file that documents Otto's standard output conventions:

1. **Banner format** (use existing pattern from otto-exec.md):
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 OTTO ► {COMMAND_NAME}
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

2. **Status symbols** (from otto-progress.md):
- ✓ completed/success
- ▶ in-progress/current  
- ○ pending/waiting
- ✗ failed/error
- ⚠ warning/attention needed

3. **Progress indicators**:
- Task progress: `━━━ Task {N}/{total}: {name} ━━━`
- Phase progress: `[████████░░░░░░░░░░░░] 40%`
- List items: `• {item}` (bullet points)

4. **Section separators**: Use blank lines, not horizontal rules mid-output.

5. **Key-value display**: `{Key}: {value}` on separate lines, indented with 2 spaces for nested items.

Keep it concise (under 50 lines). This is a reference, not enforced by code.
