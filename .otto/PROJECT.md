# Project: otto

## Vision
Otto is an autonomous code agent built on OpenCode that treats plans as executable prompts, breaking projects into focused phases and tasks while continuously capturing learnings to build institutional knowledge in your codebase.

## Goals
- Autonomous task execution — Agent completes work with minimal intervention
- Institutional knowledge — Capture learnings across sessions
- Phase-based planning — Break projects into executable phases
- Developer productivity — Speed up common development workflows
